# Qt C++ Honor's Project

## Description
This is an honors project demonstrating Qt C++ development skills and best practices.

## Features
- Qt C++ application development
- Cross-platform compatibility
- Modern C++ practices
- User-friendly interface

## Requirements
- Qt 5.15+ or Qt 6.x
- C++17 or higher
- CMake 3.16+

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Dialloni/Qt-C-Honor-s-Project.git
   cd Qt-C-Honor-s-Project
   ```

2. Build the project:
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

## Usage
Run the application:
```bash
./qt-honors-project
```
